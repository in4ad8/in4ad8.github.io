Info window:


This is a dialog/pop-up that displays specific information to a particular geometry. The data for "standard" and "multiple data layer" info windows are presented using name and value pair. The left column indicate the name and the right column indicate the value. Tapping values containing Phone Number, Website and Email will result in launching corresponding apps such as dialer, email client and web browser.


There are custom info windows created specifically for Transit, Twitter, and Instagram.
